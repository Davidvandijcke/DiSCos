# Data from (Dube 2019)

As used in the empirical application of Gunsilius (2023) .

## Usage

``` r
dube
```

## Format

### `dube`

A data frame with 652,870 rows and 3 columns:

- id_col:

  State FIPS

- time_col:

  Year

- y_col:

  `adj0contpov` variable in Dube (2019) . Captures the distribution of
  equalized family income from wages and salary, defined as multiples of
  the federal poverty threshold.
